Detailed Description

ESP32 implementation of servo control via PWM.

Controls a servo motor using LEDC PWM.

  • LOW (false) = Servo at 90° (barrier closed - vertical)
  • HIGH (true) = Servo at 0° (barrier open - horizontal)
